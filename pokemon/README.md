# Pokemon Fetcher

A simple web app that allows users to fetch and display a Pokemon's sprite based on the name they enter. This app uses the [PokeAPI](https://pokeapi.co/) to fetch Pokemon data and display the corresponding sprite.

## Features

- Input a Pokemon's name (e.g., Pikachu) to fetch its data.
- Display the Pokemon sprite based on the entered name.
- Error handling in case the entered Pokemon name is invalid.
